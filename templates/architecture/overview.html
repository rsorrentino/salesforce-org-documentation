System Architecture

Architecture Overview

The MedicalArea-Salesforce-APAC system follows a layered architecture pattern:

Presentation Layer

  • Lightning Web Components: 1614 components
  • Aura Components: Legacy UI components
  • FlexiPages: 118 pages
  • Visualforce Pages: Custom pages

Business Logic Layer

  • Apex Classes: 1977 classes
  • Apex Triggers: 58 triggers
  • Flows: 332 flows

Data Layer

  • Standard Salesforce Objects
  • Custom Objects
  • Custom Metadata Types

Integration Layer

  • REST/SOAP APIs
  • Named Credentials
  • External Services
