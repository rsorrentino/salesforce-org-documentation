Repository Structure

Directory Overview

The repository follows the standard Salesforce DX project structure:

force-app/main/default/

  • classes/ - 1977 Apex classes
  • triggers/ - 58 Apex triggers
  • lwc/ - 1614 Lightning Web Components
  • permissionsets/ - 124 Permission Sets
  • flows/ - 332 Flows
  • objects/ - Custom objects and fields
  • applications/ - Custom applications
  • namedCredentials/ - Named credentials for integrations

force-app/main/env-dependent/

  • flexipages/ - 118 FlexiPages
  • dashboards/ - Environment-specific dashboards
  • reports/ - Environment-specific reports

releases/R1/metadata/profiles/

  • profiles/ - 47 Profile definitions
